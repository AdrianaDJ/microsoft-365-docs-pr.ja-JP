# <a name="sign-up-for-microsoft-365-business"></a>Microsoft 365 Business へのサインアップ

パートナーの場合は、「microsoft[パートナーセンターから microsoft 365 Business を取得](get-microsoft-365-business.md#get-microsoft-365-business-from-microsoft-partner-center)する」を参照してください。

[microsoft 365 のビジネス上の利点を見つけ](https://www.microsoft.com/microsoft-365/business#pmg-cmp-desktop)、「microsoft 365 business で**購入**」を選択します。

1. [**ようこそ]** ページで、必要な情報を入力します。 メモ 
 
    -  選択する**国**によって、取得するサービスが決まり、後で変更することはできません。
    - **ビジネスメールアドレス**は、お客様が持っているすべてのメールであり、サブスクリプションに関する書簡を送信するために Microsoft によって使用されます。
    - [管理センター] に**会社名**が表示され、SharePoint サイトの url に使用されます。

2. [**ユーザー ID の作成**] ページで、サインインに使用するエイリアス、@ 記号の後にあるドメイン名を入力し、ドロップダウンからドメインを選択します。 これを使用してドメインを購入し、ドロップダウンから内線番号を選択できます。たとえば、 *Teresa@ContosoSkis.net*と入力すると、ウィザードによってドメインが利用可能かどうかが判断され、Microsoft 365 のビジネスコストにコストが追加されます。 新しいドメインもセットアップされ、ここで購入した場合に使用できるようになります。

    既存のドメインを使用する場合は、ドロップダウンから onmicrosoft.com を選択し*ます。* これは、後で管理センターで設定できます。
    
    ![ユーザー ID の作成ページのスクリーンショット。](media/signinuserid.png)

3. 残りのフィールドに入力して、[**アカウントの作成**] を選択します。
4. [証明します **。not'a。ロボット。** [ページ] テキストまたは通話でコードを受信するための電話番号を入力し、[**テキスト**の選択] または [**呼び出し**] を選択し、 \> **次**に確認コードを入力します。
5. [**このページを使用する場所**] で、会社の住所を入力します。 この情報は、税とサブスクリプションの利用可能性を決定するために使用されます。 [**次へ**] を選択して注文を確認し、もう一度 [**次へ**] を選択します。
6. [**支払い方法**] ページで、支払いの詳細を入力して、**注文**を行います。
    その後、[管理センター](https://go.microsoft.com/fwlink/p/?linkid=837890)に移動して[セットアップ](set-up.md)を完了します。

    > [!NOTE]
    > セットアップの最後に、[管理センター] へのリンクを選択すると、[設定](set-up.md)が直接行われます。
