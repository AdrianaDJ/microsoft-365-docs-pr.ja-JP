# <a name="microsoft-managed-desktop-roles-and-responsibilities"></a>Microsoft 管理されたデスクトップの役割と責任

組織は、管理されたデスクトップを Microsoft に登録されたと、マイクロソフト何をするのでしょうか。組織の役割は何ですか。

## <a name="microsofts-roles-and-responsibilities"></a>マイクロソフトの役割と責任

いくつかの重要な役割と責任にマイクロソフトによって提供される次のとおりです。

役割や責任 | 説明
--- | ---
MDM のポリシーの管理 | マイクロソフトは、ベスト ・ プラクティスに従って MDM のポリシーを適用し、ポリシーの変更の要求を検討してください。[デバイス ポリシー](../service-description/device-policies.md)に規定されたように、テナントには変更を加えますもします。
アプリケーションの配置 | マイクロソフトでは、Intune を使用してクライアントの Microsoft 管理されたデスクトップのデバイスへの承認されたアプリケーションを展開します。詳細については、[アプリケーション](../get-ready/apps.md)を参照してください。 
エンド ユーザー サポート | マイクロソフトは、デバイス、ウィンドウ、および Office のエンド ユーザー サポートを提供する、すべての登録ユーザーの製品群です。 
管理されたデスクトップのマイクロソフトのサービス サポート | マイクロソフトはお客様の IT にサポートを提供するが、Microsoft 管理デスクトップ運用チームから部門です。このチームは、技術的なトラブルシューティングをサポートして、変更要求、およびお客様のマイクロソフトの管理されたデスクトップ環境の問題の管理。詳細については、[サポート](../service-description/support.md)を参照してください。
セキュリティの監視 | マイクロソフトでは、Windows Defender の分析ツールを使用して顧客管理されたデスクトップのマイクロソフトのデバイスを監視します。脅威が検出された場合で、Microsoft 管理デスクトップ セキュリティ運用センター (SOC)、マイクロソフトは、お客様に通知は、デバイスを特定し、リモートで問題を修正します。詳細については、[セキュリティ](../service-description/security.md)を参照してください。
更新プログラムの監視と管理 | マイクロソフトは、Microsoft Windows および Microsoft Office の最新の品質、機能、および定義の更新プログラムがインストールされていることを確認するのには顧客の管理されたデスクトップの Microsoft デバイスを監視して積極的にします。詳細については、[更新プログラムを処理する方法](../service-description/updates.md)を参照してください。
デバイスの調達 | マイクロソフトは直接出荷エンドユーザーまたは IT の配布ポイントを選択するのに順序の管理されたデスクトップの Microsoft デバイスです。詳細については、[デバイス](../get-started/devices.md)を参照してください。

## <a name="your-roles-and-responsibilities"></a>役割と責任

下は、マイクロソフトが提供されていないが、展開を成功させるために必要な一般的なロールのセットを追加します。完全ではありませんが、ほとんどの組織の代表的です。 

ロールまたは reponsibility | 説明
--- | ---
変更管理 | Microsoft 管理されたデスクトップは、変更する必要があります、Microsoft の管理されたデスクトップ環境にするとお客様を事前に通知されます。顧客は、独自の変更管理プロセスが必要し、Microsoft 管理されたデスクトップに連絡先を設定する必要が。お客様は、リソースを確認し、これらの変更を承認する必要があります。詳細情報は、[操作と監視](../service-description/operations-and-monitoring.md)を参照してください。  
Id 管理 | ユーザー アカウント、グループにユーザーを割り当てると、メタデータを最新に保つを作成しています。 
Office 365 の構成と管理 | Exchange Oonline の管理を含みます。<br>電子メールの管理<br>メールボックスおよび規則の構成<br>交換設置管理<br><br>コラボレーション ・ ツール、SharePoint サーバーの管理、ドメイン管理、セキュリティと情報ポリシー (Microsoft 管理されたデスクトップを使用すれば Office アプリケーションはエンドユーザーのデバイスに展開して、最新情報を) Office 365 管理ポータルで設定します。 
エンド ユーザー サポート | お客様は、エンド ・ ユーザーのサポートを提供します。 <br>サイト インフラストラクチャの: すべてネットワークとインターネット接続、VPN インフラストラクチャおよびクライアントの構成、ローカルの会議室の機器、プリンター、プロキシ サーバーとファイアウォールの構成。<br><br>-会社のクラウド リソース: 電子メール、SharePoint、コラボレーション サービス、および企業全体にわたるテクノロジーのフット プリントに関連するその他のクラウド ・ インフラストラクチャです。<br><br>--業務アプリケーション: カスタム ・ ソフトウェア、サード パーティ製ソフトウェア、エンタープライズ リソース プランニング (ERP) ソフトウェア、設計ツール、およびこのドキュメントで指定されていないものです。
ユーザーおよびデバイスのグループ化 | 管理されたデスクトップの Microsoft の運用チームは作成し、IT 運用の一部として必要なデバイスとユーザー グループを管理します。お客様は、最初の接続中に IT 運用サポートされているチャネルを経由せず、これらのグループを変更しないでください。検出されたすべての変更を元に戻します。
アプリ | お客様は、(Microsoft 365 のライセンスに含まれているアプリケーション) を超える組織で使用するアプリケーションの一覧を提供します。お客様は、Appx (MSI) の可能なパッケージを提供します。<br>お客様の担当です。<br>適切な種類とライセンスの数量を持つ – アプリケーションのライセンス管理<br>アプリケーションの割り当て: Azure AD ユーザー グループにアプリケーションを割り当てる<br>– アプリケーションの更新プログラム パッケージ、およびアプリケーションの機能とセキュリティの更新の展開の監視<br>のユーザー アプリケーションのテスト (UAT)<br><br>詳細については、[アプリケーション](../get-ready/apps.md)を参照してください。
セキュリティの監視と応答 | マイクロソフト デスクトップ操作の管理の範囲外にあるセキュリティ上の問題が検出された場合、問題の重大度に基づいて優先顧客の連絡先の一覧が必要です。<br><br>お客様は、調査および以外の Microsoft 管理デスクトップ デバイスとサービスに影響を及ぼす可能性のある問題の Microsoft デスクトップ操作の管理チームに通知することを確認の問題の解決を担当します。
操作をサポートします。 | マイクロソフト デスクトップ操作の管理には、優先顧客の連絡先情報と専門家の一覧が必要があります。以外の Microsoft 管理デスクトップ運用上の問題がある場合にこれらが必要です。 <br><br>顧客は、調査および以外の Microsoft 管理デスクトップ デバイスとサービス、および、Microsoft デスクトップ操作の管理チームは常に通知を確保するためのインシデントの解決を担当します。
VPN を含むネットワーク インフラストラクチャ | セットアップ、構成および管理のトラブルシューティングとデバッグなどのネットワークに関連するすべてのインフラストラクチャとサービス、インターネット接続を含むコントロール、プロキシの構成、およびリモート接続インフラストラクチャをネットワークします。<br><br>既定では、Microsoft の管理されたデスクトップを指定したりしないプロキシを必要とします。ただし、(ハードウェアまたはソフトウェア) のいずれかを構成する場合は、プロキシで許可する必要がある Url のコレクション。お客様は、任意の競合または複数のプロキシのための非互換性のトラブルシューティングを担当します。<br><br>詳細については、[プロキシの構成](../get-ready/network.md)を参照してください。
印刷 | インストールの管理、プリンターと印刷キューを管理します。クラウド印刷が、推奨されるソリューションしますが、必須ではありません。 
モバイル デバイス | デバイスとアクセサリがマイクロソフトの管理されたデスクトップでは提供されません。Microsoft 管理されたデスクトップの運用チームは、Microsoft 管理デスクトップ デバイス、ドッキング ステーションおよび付属アクセサリのみサポートします。
