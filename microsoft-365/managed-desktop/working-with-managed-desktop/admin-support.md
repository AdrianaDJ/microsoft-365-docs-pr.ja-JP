# <a name="admin-support-for-microsoft-managed-desktop"></a>Microsoft マネージド デスクトップの管理者サポート

サポート チケットまたは Microsoft 管理されたデスクトップ管理ポータルを使用してマイクロソフトにフィードバックの要求を送信することができます。サポート リクエストは常に優先順位を送信したフィードバックします。サポート リクエストはトリアージが決定され、[重大度レベルの定義] テーブル](#sev)で説明されている重大度に従って管理されます。フィードバックを確認すると、し、応答が要求された場所を提供します。 

>[!IMPORTANT]
>確認する[、管理者の連絡先の設定](../get-started/add-admin-contacts.md)のアプリケーションのパッケージ化、デバイス、セキュリティ、およびその他の。管理者の連絡先が構成されていない場合は、これらの領域のいずれかでサポート リクエストを送信することはできます。

サポート リクエストを送信するに、管理者用ポータルの [Microsoft の管理されたデスクトップを開き **+ 新しいサポート チケット**のサポートの要求] タブでを選択します。以下の情報に役立つシナリオに応じて、選択します。重大度レベルを選択すると**想定される顧客の応答**の期待を確認してください。

サポート リクエストを作成するときは、必要なヘルプの種類に一致する種類を選択します。

サポート要求の種類 | 使用する状況
--- | ---
インシデント | Microsoft 管理デスクトップ運用チームに変更またはセキュリティ インシデントの影響が広範囲にわたるように、調査が必要です。
情報の要求 | ネットワーク、プロキシの構成、VPN システム、証明書の有効期限の変更を計画しているか、サービスに関する情報を必要なだけです。Microsoft 管理デスクトップ運用チームからの応答は、組織内での変更を通信するときを強くお勧めします。
変更要求 | Microsoft 管理デスクトップ運用チームに更新プログラムの呼び出し回数との間にデバイスを移動するなど、変更が必要です。
