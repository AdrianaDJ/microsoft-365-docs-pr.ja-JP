# <a name="configure-secure-access-to-microsoft-365-services"></a>Microsoft 365 サービスへのセキュリティで保護されたアクセスを構成する

このソリューションは、「[M365 ゴールデン config](https://aka.ms/m365goldenconfig)」に基づいています。これは、MFA、条件付きアクセス、および3層の保護 (ベースライン、機密、高機密) に関連するポリシーを実装するための規範的なガイダンスに基づいています。
