# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>メッセージがスパムとしてマークされないようにする方法

Exchange Online または Exchange Online Protection (EOP) の管理者は、サービスを通過する電子メール メッセージがスパムとしてマークされないようにできます。 これを実行するには、「 [Office 365 の安全な送信者のリストを作成](create-safe-sender-lists-in-office-365.md)する」で、更新されたバージョンのヒントと手順を参照してください。
