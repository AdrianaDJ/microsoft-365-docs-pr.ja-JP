## <a name="recover-an-inactive-mailbox"></a><span data-ttu-id="23ed8-122">非アクティブなメールボックスを回復する</span><span class="sxs-lookup"><span data-stu-id="23ed8-122">Recover an inactive mailbox</span></span>

<span data-ttu-id="23ed8-123">*InactiveMailbox*パラメーターを指定して**新しいメールボックス**コマンドレットを使用して、非アクティブなメールボックスを回復します。</span><span class="sxs-lookup"><span data-stu-id="23ed8-123">Use the **New-Mailbox** cmdlet with the  *InactiveMailbox*  parameter to recover an inactive mailbox.</span></span>


    > [!IMPORTANT]
    > <span data-ttu-id="23ed8-p105">上記のコマンドでは、 **DistinguishedName** または **ExchangeGUID** プロパティの値を使用して非アクティブなメールボックスを識別します。これらのプロパティは組織内の各メールボックスに対して一意ですが、アクティブなメールボックスと非アクティブなメールボックスとでプライマリ SMTP アドレスが等しい可能性があります。</span><span class="sxs-lookup"><span data-stu-id="23ed8-p105">In the previous command, use the value of the **DistinguishedName** or **ExchangeGUID** property to identify the inactive mailbox. 2. <span data-ttu-id="23ed8-127">次の例では、上記のコマンドで取得したプロパティを使用して、ユーザー Ann Beebe のために非アクティブなメールボックスをアクティブなメールボックスに回復します。</span><span class="sxs-lookup"><span data-stu-id="23ed8-127">This example uses the properties obtained in the previous command and recovers the inactive mailbox to an active mailbox for the user Ann Beebe.</span></span> <span data-ttu-id="23ed8-128">*Name*および*MicrosoftOnlineServicesID*パラメーターに指定する値が組織内で一意であることを確認してください。</span><span class="sxs-lookup"><span data-stu-id="23ed8-128">Be sure that the values specified for the  *Name*  and  *MicrosoftOnlineServicesID*  parameters are unique within your organization.</span></span>

    <span data-ttu-id="23ed8-129">回復された非アクティブなメールボックスのプライマリ SMTP アドレスの値は、 *MicrosoftOnlineServicesID*パラメーターで指定されたものと同じになります。</span><span class="sxs-lookup"><span data-stu-id="23ed8-129">The primary SMTP address for the recovered inactive mailbox will have the same value as the one specified by the  *MicrosoftOnlineServicesID*  parameter.</span></span>
