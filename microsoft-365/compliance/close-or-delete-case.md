Here's what happens when you close a case:</span></span>

- <span data-ttu-id="af37d-p102">ケースに保留中の任意のコンテンツの場所が含まれている場合、これらの保留が無効になります。この結果、ユーザーまたは自動プロセス (削除ポリシーなど) によってコンテンツが完全に削除または消去される場合があります。  </span><span class="sxs-lookup"><span data-stu-id="af37d-p102">If the case contains any content locations on hold, those holds will be turned off. - <span data-ttu-id="af37d-110">ケースは、セキュリティ & コンプライアンスセンターの [電子情報開示] ページに表示されたままになっています。</span><span class="sxs-lookup"><span data-stu-id="af37d-110">The case is still listed on the eDiscovery page in the Security & Compliance Center.</span></span> <span data-ttu-id="af37d-111">クローズしたケースの詳細、保持、検索、メンバーは保持されます。</span><span class="sxs-lookup"><span data-stu-id="af37d-111">The details, holds, searches, and members of a closed case are retained.</span></span>

- <span data-ttu-id="af37d-112">クローズされた case を編集できます。</span><span class="sxs-lookup"><span data-stu-id="af37d-112">You can edit a case after it's closed.</span></span> <span data-ttu-id="af37d-113">たとえば、メンバーを追加または削除したり、検索を作成したり、検索結果をエクスポートしたり、高度な電子情報開示で分析のために検索結果を準備したりできます。</span><span class="sxs-lookup"><span data-stu-id="af37d-113">For example, you can add or removing members, create searches, export search results, and prepare search results for analysis in Advanced eDiscovery.</span></span> <span data-ttu-id="af37d-114">アクティブなケースとクローズケースの主な違いは、ケースがクローズされたときに保留がオフになっていることです。</span><span class="sxs-lookup"><span data-stu-id="af37d-114">The primary difference between active and closed cases is that holds are turned off when a case is closed.</span></span>
