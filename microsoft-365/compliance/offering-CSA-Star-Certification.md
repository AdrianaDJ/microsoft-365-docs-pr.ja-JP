## <a name="microsoft-and-csa-star-certification"></a><span data-ttu-id="a89fb-112">Microsoft と CSA STAR Certification</span><span class="sxs-lookup"><span data-stu-id="a89fb-112">Microsoft and CSA STAR certification</span></span>

<span data-ttu-id="a89fb-113">Microsoft Azure、Microsoft Intune、Microsoft Power BI は、STAR Certification を取得しています。STAR Certification には、クラウド プロバイダーのセキュリティ態勢について、独立した第三者による厳格な評価が必要です。</span><span class="sxs-lookup"><span data-stu-id="a89fb-113">Microsoft Azure, Microsoft Intune, and Microsoft Power BI have obtained STAR Certification, which involves a rigorous independent third-party assessment of a cloud provider’s security posture.</span></span> <span data-ttu-id="a89fb-114">この STAR 認定は、ISO/IEC 27001 認定を取得しているか、また、CCM において規定されている基準を満たしているかで判定されます。</span><span class="sxs-lookup"><span data-stu-id="a89fb-114">STAR Certification is based on achieving ISO/IEC 27001 certification and meeting criteria specified in the CCM.</span></span> <span data-ttu-id="a89fb-115">これは、クラウド サービス プロバイダーが、該当する ISO/IEC 27001 の要件に準拠していること、CCM において規定されているクラウド セキュリティを左右する問題を解決していること、および、CCM のコントロール領域に含まれるアクティビティの管理について STAR Capability Maturity Model (STAR 能力成熟度モデル) を基に評価されていることを示します。</span><span class="sxs-lookup"><span data-stu-id="a89fb-115">It demonstrates that a cloud service provider conforms to the applicable requirements of ISO/IEC 27001, has addressed issues critical to cloud security as outlined in the CCM, and has been assessed against the STAR Capability Maturity Model for the management of activities in CCM control areas.</span></span>  
  
<span data-ttu-id="a89fb-116">評価中に、公認の CSA 認定監査人が、16 ある CCM のコントロール領域の 1 つ 1 つに Maturity Capability (能力成熟度) スコアを割り当てます。</span><span class="sxs-lookup"><span data-stu-id="a89fb-116">During the assessment, an accredited CSA certification auditor assigns a Maturity Capability score to each of the 16 CCM control areas.</span></span> <span data-ttu-id="a89fb-117">その後、スコアの平均を基に、全体的な成熟度レベルを割り当て、「ブロンズ」、「シルバー」、「ゴールド」のうち対応するアワードが授与されます。</span><span class="sxs-lookup"><span data-stu-id="a89fb-117">The average score is then used to assign the overall level of maturity and the corresponding Bronze, Silver, or Gold award.</span></span> <span data-ttu-id="a89fb-118">Azure、Intune、Power BI、Microsoft Cloud App Security は、ゴールド レベルで Cloud Security Alliance (CSA) STAR Certification を授与されています。</span><span class="sxs-lookup"><span data-stu-id="a89fb-118">Azure, Intune, Power BI, and Microsoft Cloud App Security were awarded Cloud Security Alliance (CSA) STAR Certification at the Gold level.</span></span>
