# <a name="cortana-in-office-365"></a>Office 365 の Cortana

個人の生産性アシスタントである Cortana は、時間を節約し、最も重要なことに注目するために、AI によるエクスペリエンスを提供します。 Cortana を使用すると、ユーザーは仕事と生活の両方で自分の毎日の生産性を向上させることができます。 有効な職場または学校のアカウントを使用してサインインすると、ユーザーは Office 365 生産性ツール内でクラウドベースの Cortana アシスタントサービスを利用できるようになります。 Office 365 の Cortana は、Office 365 のエンタープライズレベルのプライバシー、セキュリティ、およびコンプライアンスの約束を満たします。

個人の生産性アシスタントとして、Cortana は、電子メール、ファイル、チャットなどの Office 365 データを安全かつ安全に処理および理由を提供するように設計されており、時間を節約し、効率を高め、ユーザーの生産性を向上させることができます。 

次に、エンタープライズの生産性に Cortana を焦点を置いています。

- 他の Office 365 サービスとの一貫性のある Office 365 の Cortana は、[オンラインサービスの利用規約 (OST)](https://www.microsoft.com/en-us/licensing/product-licensing/products)に反映されているのと同じエンタープライズレベルのプライバシー、セキュリティ、およびコンプライアンスの約束を満たします。

- ブリーフィングメールや電子メールの再生など、新しいクラウドベースの Cortana アシスタントサービスは、Office 365 の Cortana を使用して有効にし、それらの約束に完全に準拠します。 これらの機能は、米国内にメールボックスを持つユーザーのために英語 (米国) の市場で利用可能になっており、米国のメールボックスを持つ多国籍企業の従業員にも使用できます。

- ユーザーは、Outlook Mobile などのクライアントアプリケーションを使用して、個別のライセンス条項に従って、Office 365 サービスの Cortana に接続することができます。

- Windows 1909 10 の Cortana、iOS および Android に cortana アプリを含む既存のコンシューマーエクスペリエンスは、 [Microsoft サービス規約](https://www.microsoft.com/en-us/licensing/product-licensing/products)と[microsoft のプライバシー](https://privacy.microsoft.com/en-us/privacystatement)に関する声明で規定されています。 これらの用語は、コンシューマー資格情報を使用してサインインしたときにユーザーに提供される Office 365 サービスの Cortana も管理します。

## <a name="what-data-is-processed-by-cortana-in-office-365"></a>Office 365 で Cortana によって処理されるデータ

Office 365 の Cortana は、ユーザーからのクエリ、ユーザーの要求を満たすために必要な Office データ、およびサービスを実行するために Microsoft システムによって生成されたその他のテレメトリを処理します。 Office 365 で Cortana によって収集されるデータには、ユーザーの音声クエリのテキスト表現 (音声認識からのトランスクリプション) が含まれています。 このテキストデータは顧客データであり、[オンラインサービス](https://www.microsoft.com/en-us/licensing/product-licensing/products)の使用条件に従って管理されます。 これは、[オンラインサービスの用語](https://www.microsoft.com/en-us/licensing/product-licensing/products)と一貫性のある機械学習モデルを開発および改善するためにのみ使用されます。

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-office-365"></a>Office 365 の Cortana での顧客データのガバナンスモデルとは

他の Microsoft 365 および Office 365 サービスとの一貫性。 Office 365 の Cortana はセキュリティで保護され、[オンラインサービス](https://www.microsoft.com/en-us/licensing/product-licensing/products)の使用条件に従っています。 これには、偶発的な損失、変更、許可されていない開示またはアクセス、または不正な破壊からお客様のデータを保護するための一連の約束が含まれています。 お客様のデータも、厳格なアクセス制限の対象となります。 Microsoft では、顧客データのみを使用して、同意されたサービスと、それらのサービスとの互換性を確保しています。 詳細については、次の表を参照してください。

## <a name="cortana-in-office-365-features"></a>Office 365 の Cortana の機能

### <a name="play-my-emails"></a>自分のメールを再生する

「Outlook mobile を介して接続された電子メールを再生する」は、ユーザーが電話、ヘッドホン、または接続されたオーディオデバイスのスピーカー経由で受信トレイを聞くことができる、音声駆動型のハンズフリー操作です。 ユーザーは、Cortana に最近の電子メールを読み上げてもらい、フラグ、アーカイブ、削除、スキップのメッセージなどのアクションを実行するように cortana に依頼できます。 この機能は、commuting 中に、または外出中にメールを取得する場合に特に役立ちます。 ユーザーが電子メールを再生するときに Cortana と話をすると、音声音声要求は Office 365 の Cortana に直接送られます。 ユーザーの電子メールの読み上げ機能は、Office 365 cloud 内部で処理されます。 このプロセスでは、ユーザーのモバイルデバイスで Office 365 データは処理されず、新しい電子メールデータは保存されません。 音声コマンドのトランスクリプト (「開封済み」、「次」、「フラグ」など) は、[オンラインサービスの用語](https://www.microsoft.com/en-us/licensing/product-licensing/products)のデータ保護条件に従って保持できます。

ユーザーが再生を一時停止したり、次のメッセージにスキップしたりするのに十分な時間を与えるために、電子メールが保護されているときに、Cortana は、メッセージを読む前に一時的に一時停止します。 プライベート電話呼び出しと同様に、機密情報が overheard される可能性がある場所で再生を開始するときには注意が必要です。 このような場合は、「Outlook mobile で電子メールを再生する」を使用して、組織の従業員が適切な環境でヘッドフォンを磨耗させることをお勧めします。

### <a name="opt-out-of-play-my-emails"></a>オプトインのメールを再生する

次の手順を使用して、個人が電子メールを再生することをオプトアウトできます。

1. Outlook mobile を開きます。

2. [**設定**] に移動します。
  
3. [**メールを再生**する] を選択します。

4. 無効にするアカウントの切り替えを [オフ] に移動します。

### <a name="briefing-email"></a>ブリーフィング電子メール

Cortana は、作成したタスクやコミットメントを使用して個人用のブリーフィングメールを送信します。この電子メールには、**実行**したことをマークしたり、フォーカス時間をスケジュールして完了することができます。 また、会議の概要と、自分の日の関連ドキュメントも含まれています。 Cortana は、ユーザーの Office 365 メッセージから情報を抽出し、それをブリーフィングメールに統合するまで Office 365 メールボックスに格納します。 Office 365 メールボックスの外部で個人データにアクセスできるとは限りません。

### <a name="how-to-opt-out-of-briefing-email"></a>ブリーフィングメールをオプトインする方法

管理者は、Microsoft 365 管理センターで[トグル](https://docs.microsoft.com/briefing/be-admin)を使用して、組織のブリーフィングを構成できます。 個人は、メッセージのフッターで [購読取り消し] を選択することで、Cortana のブリーフィングメールから脱退できます。

組織の生産性を向上させるために、Office 365 の機能により多くの Cortana を引き続き導入します。

[Microsoft コンプライアンスサービス](https://docs.microsoft.com/microsoft-365/compliance/offering-home)の詳細をご覧ください。

## <a name="how-is-cortana-in-office-365-different-from-other-cortana-features-i-may-have-previously-experienced"></a>Office 365 の Cortana は、以前に経験した他の Cortana 機能とどのような違いがありますか?

企業での Cortana の動作を考える2つの方法を次に示します。

**365 office の cortana を使用する組織のための新しいエクスペリエンス**: office 365 サービスの cortana は、組織のセキュリティとコンプライアンスのニーズを満たすように設計されています。

1. これは新しいサービスです。このドキュメントでは、ここで説明します。

2. オンラインサービスの利用規約の対象となるサービスについては、microsoft がデータプロセッサを使用しています。 Microsoft は、お客様によって要求されたオンラインサービスを提供し、お客様によって指示された目的のために、お客様からの顧客データを収集して使用し EU の一般データ保護規則 (GDPR) の下で、お客様はデータのデータコントローラーです。 商用クラウドのお客様には、[オンラインサービス](https://www.microsoft.com/en-us/licensing/product-licensing/products)の使用条件と[プライバシーに関するより多くの透明性を紹介](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)しています。

3. 例として、電子メールを再生することは、ユーザーが Outlook を使用して iOS 用の Outlook 経由で接続し、Office 365 で Cortana を利用できる Cortana サービスです。

4. 管理者は、Office ProPlus アプリケーションを使用しているときにオプションの接続エクスペリエンスと同様に、Cortana のオプションの接続エクスペリエンスに関するコントロールを常に備えています。

**コンシューマー向けの既存のサービス**: Cortana オプション接続サービスは、主にコンシューマーエクスペリエンスを目的として設計されており、Windows 10 (バージョン1909以前) および IOS および Android 上の Cortana アプリで現在提供されています。

1. これらのエクスペリエンスは、天気、ニュース、トラフィックなどの機能を有効にします。

2. テナント管理者は、Windows 10 (バージョン1909以前) と iOS および Android 上の Cortana アプリが、Cortana が Office 365 テナントデータに接続することを許可するかどうかを制御できます。

組織の Microsoft ホストされているデータへの Cortana アクセスをオフにする

1. Microsoft 365 管理センターで、[**設定** > **サービス**] を選択し、[ **Cortana**] を選択します。

2. [次の Cortana での**接続エクスペリエンスを許可**する] チェックボックスをオフにして、cortana の接続エクスペリエンスを無効にします。

3. [**変更の保存**] を選択します。
