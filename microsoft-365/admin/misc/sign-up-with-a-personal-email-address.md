# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>個人用メールアドレスを使用した Office 365 へのサインアップ

## <a name="when-you-skip-adding-a-custom-domain"></a>カスタムドメインの追加をスキップした場合

  
個人の電子メールアドレスを使用して Office 365 にサインアップして、 [カスタムドメイン](../get-help-with-domains/what-is-a-domain.md) の追加をスキップした場合は、特定の機能へのアクセスを制限することになります。 
  
ここでカスタムドメインの追加を選択すると、以下のすべてのプレミアム機能にアクセスできます。
  
|**機能**|**Office 365 と個人用電子メール**|**Office 365 とカスタムドメイン**|
|:-----|:-----|:-----|
|**OneDrive <sup>1、2</sup>**| [個人用 OneDrive ](https://onedrive.live.com/about/en-us/plans/)| [OneDrive for Business](https://onedrive.live.com/about/en-us/business/) |
|**Office アプリケーション: Word、Excel、PowerPoint、OneNote、Outlook、Access (PC のみ)、**| はい | はい 
|**ビジネスアプリケーション <sup>3</sup> : Microsoft Connections、請求書、リスト、予約、マイル、および Outlook カスタマーマネージャー**| いいえ | はい 
|**Microsoft 365 管理センターへのアクセス**| 制限付きアクセス (課金、サポート、およびドメインのセットアップ) | はい
|**ユーザーを追加する**| いいえ | はい
|**Office 365 サービス: Exchange Online、Skype for Business、Microsoft Teams、SharePoint Online**| いいえ | はい
|**セキュリティおよびコンプライアンスツール**| いいえ | はい
