# <a name="meetings--people-experiences"></a>会議–ユーザーエクスペリエンス

生産性スコアは、Microsoft 365 とそれをサポートするテクノロジエクスペリエンスを利用して、組織のデジタル変換に関する洞察を提供します。 組織のスコアは、人とテクノロジの実績を反映しており、組織と同じようなベンチマークと比較できます。 [会議] カテゴリは、ユーザーエクスペリエンス対策の一部です。 詳細については、「 [生産性スコアの概要」](productivity-score.md) と「 [Microsoft のプライバシー](https://privacy.microsoft.com/privacystatement)に関する声明」を参照してください。

## <a name="why-your-organizations-meetings-score-matters"></a>組織の会議スコアが重要な理由

会議 (アイデアの調査、計画、問題の解決、意思決定を担当する) は、組織の生産性の基本的な柱となります。 調査では、ユーザーがオンライン会議ツールを効果的に使用している場合、週に最大で104分の時間を節約できることが示されています。 [証拠](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf)を参照してください。

## <a name="how-we-calculate-the-meetings-score"></a>会議スコアの計算方法

ここでは、このカテゴリの主要な指標が含まれている経験について、主な洞察を提供しています。 次に、これらの指標について、組織のスコアを計算するために、次に示すスコアリングフレームワークを使用します。

### <a name="primary-insight"></a>主要な洞察

Microsoft Teams は、Outlook の予定表と統合されており、会議をより魅力的かつ効果的にするための機能のホストを提供します。 主要な洞察は、組織内で開催されたすべての Microsoft Teams オンライン会議を調べます。 また、主な洞察は、Teams 会議のベストプラクティスの少なくとも1つに従う会議の数も追跡します。

:::image type="content" source="../../media/primaryinsights-meetings.png" alt-text="ベストプラクティスを使用した会議の主要な洞察。":::

1. **ヘッダー:** 会議中にビデオや画面を共有していた、過去28日間に開催された Microsoft Teams のオンライン会議の割合を示します。
2. **本文:** ビデオや画面共有の使用など、会議中のベストプラクティスに従うことで、会議の効率を高める方法について説明します。
3. **視覚エフェクト (現在の状態):**

      - この横棒グラフでは、青 (色付き) の部分は、ヘッダーに表示されるパーセンテージを表します。
      - 分数 (分子/分母) は、ヘッダーに表示されるパーセンテージを計算するために使用されます。
         - 分子: ビデオまたは画面の共有を使用した組織のユーザーを含む、オンラインの Microsoft Teams 会議の数。
         - 分母: 過去28日間に開催された組織の人物を含むオンラインの Microsoft Teams 会議の数。
      - 重要な指標のピアベンチマーク値もパーセンテージとして表示されます。
1. **リソースへのリンク:** ヘルプコンテンツを表示するには、このリンクを選択します。

次のグラフは、主な洞察からの主要な指標の分子と分母の両方の傾向線を示しています。 言い換えれば、ビデオや画面の共有などのベストプラクティスを備えたオンラインの Microsoft Teams 会議の数と、過去180日間にわたって開催されたオンラインの Microsoft Teams 会議の合計数を示すことができます。 折れ線グラフの各データ要素は、過去28日間のアクティビティの集計です。

### <a name="scoring-framework"></a>スコアリングフレームワーク

組織の会議スコアは、組織内のオンライン Microsoft Teams 会議が過去28日以内にベストプラクティスに従う度合いを測定します。 これは、会議に出席した組織内のユーザー数と、会議の期間に基づいて重み付けされます。

## <a name="explore-more-about-meetings-in-your-organization"></a>組織内の会議の詳細を調べる

また、組織内のユーザーがどのように会議を開催しているかを理解するための、hep へのサポート情報も提供しています。 これらの追加指標は、生産性スコアに直接貢献するものではありませんが、デジタル変換の一部としてアクションプランを作成するのに役立ちます。

### <a name="breakdown-of-how-many-meetings-follow-best-practices"></a>ベストプラクティスに従った会議の数の内訳

:::image type="content" source="../../media/videouse-meetings.png" alt-text="会議の主な見解を示す図。ユーザーエクスペリエンス。":::

1. **ヘッダー:**  主要な洞察と採点で検討されているビデオのベストプラクティスを使用する、オンラインの Microsoft Teams 会議の割合を強調しています。
2. **本文:** 会議中にこれらのプラクティスを使用して、より魅力的なものにするための価値に関する hrovides 情報。 
3. **視覚エフェクト:** 視覚エフェクトのブレークダウンは、オンラインの Microsoft Teams 会議が以下のベストプラクティスに従っている程度を表すことを目的としています。 
    - **ビデオ**: 色の部分と分数は、ビデオが有効になっているオンラインの Microsoft Teams 会議の割合を表しています。 分数は次のように構成されています。 
        - 分子: 過去28日間に開催され、少なくとも1人の参加者がオンにしたビデオを含むオンラインの Microsoft Teams 会議。 
        - 分母: 過去28日間に組織内に開催されたオンラインの Microsoft Teams 会議の合計数 
    - **画面の共有**: 色の部分と分数は、ユーザーが画面共有機能を使用したオンラインの Microsoft Teams 会議の割合を表します。 分数には次のものが含まれます。 
        - 分子: 過去28日間に開催され、少なくとも1人の参加者が画面を共有していたオンラインの Microsoft Teams 会議。
        - 分母: 過去28日間に組織内で実施されたオンラインの Microsoft Teams 会議の合計数


### <a name="distribution-of-time-spent-by-people-in-meetings"></a>会議中のユーザーによって費やされた時間の配分

:::image type="content" source="../../media/percentageofpeopleusingteams.png" alt-text="20時間以上に Teams 会議に出席するユーザーの割合を示す図。":::

1. **ヘッダー:** 過去28日間のアクティビティに基づいて、1週間に20時間以上オンラインの Microsoft Teams 会議に参加する組織内のユーザーの割合を示します。
2. **本文:** 会議のベストプラクティスを使用して会議を参加させ、生産的なものにすることの価値についての詳細を提供します。
3. **視覚エフェクト:** 過去28日以内の会議で毎週かかった時間に基づいて、組織内のユーザーの表示を提供します。 カテゴリごとに、次の情報が提供されます。
      - **会議参加者の合計数:** 過去28日間のミーティング期間の平均の範囲に基づいて、ミーティングを参加した組織内のユーザーの数を表示します。 たとえば、6-10 hours カテゴリは、過去28日間の1週間あたりの平均時間について、会議を出席したユーザーの数を示します。
      - **ビデオを使用した会議の参加者:** カテゴリごとに、組織内のどのユーザーが過去28日以内にビデオを使用して会議中だったかが表示されます。
      - **画面共有を使用した会議の出席者:** カテゴリごとに、過去28日以内に画面共有が含まれていた会議中のユーザー数を示します。

### <a name="distribution-of-meeting-length-by-type"></a>種類別の会議の長さの分布

:::image type="content" source="../../media/distribution-meetinglength.png" alt-text="会議の長さの分布を示すグラフ。":::

1. **ヘッダー:** 30分以内に過去28日間のインスタント (未スケジュール) オンラインの Microsoft Teams 会議の割合を示します。
2. **本文:** 即時会議を使用して問題を迅速に解決する価値に関する情報を提供します。
3. **視覚エフェクト:** 過去28日以内に組織内で実施されたインスタントおよび予約された会議の長さ (分単位) を指定します。 Distribution は、各会議を1-15 分、16-30 分、31-60 分、および60分以上に分類します。

    > [!NOTE]
    > 予約された会議には、ユーザーの予定表に表示されたすべての会議が含まれます。 このインスタント会議には、1:1 とグループ通話の両方の呼び出しと、 &quot; Microsoft Teams チャネルの [今すぐ会議] 機能を使用して開始された会議が含ま &quot; れます。


### <a name="use-of-different-meeting-types"></a>さまざまな種類の会議の使用

:::image type="content" source="../../media/percentparticipation-meetingtypes.jpg" alt-text="さまざまな種類の会議の参加率を示す図。":::

1. **ヘッダー:** 30分未満の期間を超える、過去28日間のインスタントオンライン Microsoft Teams 会議の割合を強調表示します。
2. **本文:**&quot;Microsoft Teams チャネル機能の [Meet now (即時会議) の値に関する情報を提供し &quot; ます。
3. **視覚エフェクト:** オンラインの Microsoft Teams 会議に参加している人が参加している会議の種類を示します。 各会議の種類は横棒で表され、色付きの部分と分数は次のようになります。
    - **インスタント1:1 通話**:
        - 分子: 過去28日以内に1:1 呼び出しに出席したユーザーの数
        - 分母: 過去28日間にオンラインの Microsoft Teams 会議に出席したユーザーの数
   - **インスタントグループ通話**:
        - 分子: 過去28日以内にグループ通話を出席したユーザーの数
        - 分母: 過去28日間にオンラインの Microsoft Teams 会議に出席したユーザーの数
   - **チャネル内の即時会議**:
        - 分子: Microsoft Teams チャネル内で即時会議機能を使用しているユーザーの数 &quot; &quot; (即時会議の場合)、過去28日以内
        - 分母: 過去28日間にオンラインの Microsoft Teams 会議に出席したユーザーの数
    - **スケジュールされた1回限りの会議:**
        - 分子: 過去28日以内に予定表の1回限りのオンライン Microsoft Teams 会議に出席したユーザーの数
        - 分母: 過去28日間にオンラインの Microsoft Teams 会議に出席したユーザーの数
    - **定期的な会議のスケジュール:**
        - 分子: 過去28日以内に定期的な会議のインスタンスに対して予定表 (スケジュール済み) に出席したユーザーの数
        - 分母: 過去28日間にオンラインの Microsoft Teams 会議に出席したユーザーの数
