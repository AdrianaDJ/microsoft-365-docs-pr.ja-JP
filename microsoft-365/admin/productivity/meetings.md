# <a name="meetings--people-experiences"></a>会議–ユーザーエクスペリエンス

生産性スコアは、ユーザーが Microsoft 365 アプリを使用する方法と、テクノロジエクスペリエンスをサポートする方法について測定基準を使用して作業を行う方法を変革するのに役立ちます。 このスコアは、ユーザーとテクノロジエクスペリエンスの組織&#39;のパフォーマンスを反映し、自分のスコアを自分のような組織と比較します。 [会議] カテゴリは、ユーザーエクスペリエンス内の1つの指標です。 詳細については、「 [生産性スコアの概要」](productivity-score.md)を参照してください。

## <a name="why-your-organizations-meetings-score-matters"></a>組織の会議のスコアが重要な理由

会議 (アイデアの調査、計画、問題の解決、意思決定を担当する) は、組織の生産性の基本的な柱となります。 調査では、ユーザーがオンライン会議ツールを効果的に使用している場合、週に最大で104分の時間を節約できることが示されています。 [証拠](https://vc2prod.blob.core.windows.net/vc-resources/TEIStudies/TEI%20of%20Microsoft%20365%20E5%20-%20Oct%202018.pdf)を参照してください。

## <a name="how-we-calculate-the-score"></a>スコアの計算方法

ここでは、このカテゴリの主要な指標が含まれている経験について、主な洞察を提供しています。 次に、スコアを計算するためにこれらの指標として、次に示すスコアリング framework を使用します。

Microsoft Teams は、Outlook の予定表と統合されており、会議をより魅力的かつ効果的にするための機能のホストを提供します。 主要な洞察は、組織内で開催されたすべての Microsoft Teams オンライン会議を調べます。 また、主な洞察は、Teams 会議のベストプラクティスの少なくとも1つに従う会議の数も追跡します。

2. **本文:** ビデオや画面共有の使用など、会議中のベストプラクティスに従うことで、会議の効率を高める方法について説明します。
3. **視覚エフェクト (現在の状態):**

      - この横棒グラフでは、青 (色付き) の部分は、ヘッダーに表示されるパーセンテージを表します。
      - 分数 (分子/分母) は、ヘッダーに表示されるパーセンテージを計算するために使用されます。

### <a name="scoring-framework"></a>スコアリングフレームワーク

会議スコアは、組織内のオンラインの Microsoft Teams 会議が、過去28日以内にベストプラクティスに従う度合いを測定します。 この値は、会議に参加している組織内のユーザー数と、会議の期間に基づいて重み付けされます。

## <a name="explore-more-about-meetings-in-your-organization"></a>組織内の会議の詳細を調べる

また、組織内のユーザーがどのように会議を開催するかを理解するためのサポート情報も提供します。 これらの追加の測定基準は、生産性のスコアに直接貢献する&#39;はありませんが、ユーザーが作業する方法への最適化をサポートするためのアクションプランを作成するのに役立ちます。

## <a name="people-in-your-organization"></a>組織内のユーザー

このセクションでは、ページ上のすべての分析情報に関連するアクティビティのユーザーレベルの詳細を提供することによって、重視する指標を操作するのに役立つ情報を提供します。

ユーザーレベルの表では、次の列が示されています。

- **Username**  は、ユーザーの電子メールアドレスです。
- [ **表示名** は、ユーザーの完全な名前です。
- [ **最後のアクティビティの日付** は、過去28日以内に会議内のいずれかの insights に関連するアクティビティをユーザーが最後に実行した日時です。
- **Department:** Azure Active Directory (AAD) でユーザーに対して定義されている部署
- **会社:** AAD でユーザーに対して定義されている会社
- **国:** AAD でユーザーに対して定義された国
- **状態:** AAD でユーザーに対して定義されている状態
- **市区町村:** AAD でユーザーに対して定義された市区町村
- **会議の参加** : 過去28日間のオンライン Microsoft Teams 会議の数
- **画面共有が参加している会議:** 過去28日以内に画面を共有していたオンラインの Microsoft Teams 会議の数
- **ビデオが出席される会議:** 過去28日以内にビデオがあったオンラインの Microsoft Teams 会議の数
- **会議の時間:** 過去28日以内に参加したオンラインの Microsoft Teams 会議に費やした時間数
- **参加したインスタント会議:**&quot; &quot; 過去28日以内に参加した、チャネル会議に1:1 通話、グループ通話、会議を参加させることができるインスタント会議の数
- 予定された **1 回限りの会議出席依頼:** 過去28日以内に実行された予定表の予定の1回限りの会議の数
- **参加した定期的な会議のスケジュール:** 過去28日以内に有人で実行された予定表の定期的な会議のインスタンスの数
- **インスタント1:1 通話は次のとおりです。** 過去28日以内に参加した1:1 通話の数 (即時会議の一部)
- **インスタントグループの通話の参加:** 過去28日以内に参加したグループ通話の数 (即時会議の一部)
- **チャネル内の即時会議:**&quot; &quot; 過去28日以内に参加した、チャネル内の現在の会議数 (即時会議の一部)

## <a name="notes-and-caveats"></a>メモと警告

- オンラインの Microsoft Teams 会議へのすべての参照に &quot; &quot; は、組織のユーザーに関係する、予約された会議とインスタント会議 (1:1 通話、グループ通話、チャネルでの即時会議) の両方が含まれます。
- 生産性スコアインサイト用の会議と通話には、次のような特徴があります。
  - 期間が1分を超えています
  - 少なくとも2人のユーザーが参加する
- Microsoft Teams へのアクセス権を持つユーザーのみが評価の対象となります。
- ブロードキャストイベントと、会議または通話としてタグ付けされていない特定のイベントを除外します。
- 参加者間の画面共有で開始されるイベントは、1:1 通話 (参加者2人) またはグループ呼び出し (3 人以上の参加者) として分類されます。
