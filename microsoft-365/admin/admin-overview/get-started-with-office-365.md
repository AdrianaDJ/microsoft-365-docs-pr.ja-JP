開始する際に役立つリソースを以下に示します。
  
- [管理者向けのトレーニング](https://support.office.com/article/e990f8ff-56d7-450e-ad9f-74ae8718ef09.aspx): 管理者向けの無料のオンラインクラスへのリンクを取得します。
    
- [管理者ヘルプ](https://support.office.com/article/17d3ff3f-3601-466e-b5a1-482b31cfb791.aspx): 主なタスク、管理ツールの概要、および問題のトラブルシューティングに関する情報を取得します。
    
- [サポートへのお問い合わせ-管理者向けヘルプ](../contact-support-for-business-products.md): 電話をかけて、オンライン要求を開くか、コミュニティのヘルプをご利用ください。
